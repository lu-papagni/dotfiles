# Dotfiles

> [!IMPORTANT]
> On Unix systems make sure these dependencies are installed:
> - curl
> - tar
> - gzip

Quick install:

```bash
# Unix
sh -c "$(wget -qO- get.chezmoi.io/lb)" -- init --apply 'lu-papagni'
```

```ps1
# Windows
iex "&{$(irm 'https://get.chezmoi.io/ps1')} -b ~/.local/bin"
```
